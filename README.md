# ai-lab

Local-first AI coding and trading system.

## Operating model

- Develop locally in Ubuntu/WSL
- Use Windows as the operator desktop and control surface
- Store code, docs, configs, and workspace memory in Git
- Store meaningful large outputs and durable artifacts in Azure later
- Keep the laptop replaceable
